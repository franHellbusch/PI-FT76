# Colorfly Studio

Generador de paletas de colores interactivo. Proyecto del Modulo 1 del bootcamp.

## Tecnologias
- HTML5 semantico
- CSS3 (Flexbox, Grid, variables y colores HSL)

## Como usar
Abri `index.html` en el navegador.
